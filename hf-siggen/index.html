How to install in your email client

Gmail (web)

  1. Click "Copy Signature (Recommended)" above.
  2. In Gmail, click the gear icon (top right) → See all settings.
  3. Stay on the General tab and scroll down to the Signature section.
  4. Click + Create new, give it a name (e.g., "Highlands Fellowship"), and click Create.
  5. Click inside the signature editor and paste (Ctrl/Cmd+V).
  6. Under Signature defaults, set it as the default for new emails and replies/forwards.
  7. Scroll to the bottom of the page and click Save Changes.

Gmail on mobile (iOS/Android app) only supports plain-text signatures in its settings. Your HTML signature will still appear automatically when composing on the web — for mobile, the simplest option is to compose in a browser tab on your phone.

Outlook (desktop — Windows or Mac)

  1. Click "Copy Signature (Recommended)" above.
  2. Windows: File → Options → Mail → Signatures.
    Mac (new Outlook): Outlook → Settings → Signatures.
    Mac (legacy Outlook): Outlook → Preferences → Signatures.
  3. Click New (or + on Mac), name the signature.
  4. Paste into the edit box (Ctrl/Cmd+V).
  5. Set as default for new messages and replies. Save / OK.

Outlook desktop ignores dark-mode CSS, so it always renders the light version — this is expected and looks correct.

Microsoft 365 / Outlook on the web

  1. Click "Copy Signature (Recommended)" above.
  2. Settings (gear, top right) → Mail → Compose and reply.
    (If you don't see it, click "View all Outlook settings" at the bottom of the settings pane.)
  3. Under Email signature, click + New signature, give it a name.
  4. Paste into the editor (Ctrl/Cmd+V).
  5. Under Select default signatures, set it for new messages and replies/forwards. Save.

Apple Mail (macOS)

  1. Click "Copy Signature (Recommended)" above.
  2. Mail → Settings (or Preferences on older versions) → Signatures tab.
  3. Select your account in the left column, click + to add a new signature, name it.
  4. Click into the right-hand editor pane and paste (Cmd+V).
  5. Important: uncheck "Always match my default message font" at the bottom — otherwise Mail strips the formatting.
  6. Drag the new signature onto your account if it isn't already attached, and pick it from the "Choose Signature" dropdown.

Apple Mail (iOS / iPadOS)

iOS Mail doesn't have a rich signature editor in Settings — it accepts plain text only there. To get the formatted signature in, use this workaround:

  1. On your iPhone/iPad, open this page in Safari and fill in your info.
  2. Tap and hold inside the light-mode preview above, then drag the selection handles to cover the entire signature (logo through website).
  3. Tap Copy.
  4. Open the Mail app and compose a new message to yourself. Paste the signature into the body and send it.
  5. Open the email you just sent. Tap and hold on the signature, Select All, then Copy.
  6. Go to Settings → Mail → Signature. Tap into the signature field, paste, and before tapping out shake the device (or tap and hold → Undo / Undo Attributes) — this preserves the formatting that iOS otherwise strips on first paste.
  7. If you have multiple accounts, tap Per Account at the top and paste into each.

The "shake to undo attributes" step keeps the rich formatting — without it, iOS converts the paste to plain text.
